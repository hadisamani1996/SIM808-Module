```
connecting AD232Sensor to Arduino-Uno and send data to server with SIM808
```
requirement:
1. Arduino-Uno board
2. SIM808 Module
3. Server Account
4. AD232 Sensor
***
**ow to use:**

__onnect Pins as below:__

| Arduino_Uno-Pin | SIM808 | AD232Sensor    |
|-----------------|:------:|------------:|
|     7           | RX-Pin  |               |
|     8           | TX_Pin  |               |
|     10          |         |   L0+         |
|     11          |         |   L0-         |
| GND             | GND     | GND           |
| 3V3             | 4V0     | 3V3           |
